# NewKernel and NewBoot

You need:

- MinGW for the kernel and bootloader.
- Netwide Assembler to output COFF object code.

Start by cloning the repo:

```
git clone git@github.com:Mahrouss-Logic/h-core.git
```

And execute:

```
make h-core-<cpu>-<hardware>
```
